# writing-that-works

A Claude Code / Cowork skill that helps draft and edit business prose that gets results — emails, memos, business letters, presentations and decks, plans, reports, recommendations, proposals, fund-raising and sales letters, resumes, cover letters, and speeches.

Distilled from Kenneth Roman & Joel Raphaelson's *Writing That Works* (3rd ed.).

## What's inside

- `skills/writing-that-works/SKILL.md` — the full skill
- `.claude-plugin/plugin.json` — plugin manifest

## When the skill triggers

Use this whenever you're asked to write or improve any of the formats above, or to be more direct, less jargony, or more persuasive in business writing.

It does **not** trigger for tweets, social posts, marketing copy, fiction, or UX microcopy — those have different rules.

## Install in Cowork

Settings → Plugins / Marketplaces → Add marketplace → paste this repo's URL → install `writing-that-works`.
